# AutoNewsDoc
**Тестовое задание:**
_Разработать приложение, отображающее новостную ленту._

## About

![alt text](https://github.com/AndNasPlay/AutoNewsDoc/blob/main/ReadmeFiles/readmeMain.jpg "screen")

The application is designed to retrieve information about the news feed via API and display it on the screen.

![alt text](https://github.com/AndNasPlay/AutoNewsDoc/blob/main/ReadmeFiles/readmeBottom.jpg "screen")

## Features
Swift Programming Language 
Architecture: MVVM + Combine, UICollectionView + CompositionalLayout + DiffableDataSource UIKit, URLSession, async / await

## Requirements

Xcode 26.4 
iOS 26.2
